# OMG - Vue based admin template

## Node version
```
Node v16.18.1
```

## Git repo
```
https://github.com/andresgiraldo117/Andres_Giraldo_OMG_frontend.git
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
